## Task 1: Fix icon loading + add animation keyframes to globals.css

**Files:**
- Modify: `app/globals.css`
- Modify: `app/layout.tsx`

- [ ] **Step 1: Remove `@import` font lines from globals.css**

In `app/globals.css`, replace lines 5–6:
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=Manrope:wght@700;800;900&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:wght,FILL@100..700,0..1&display=swap');
```
with nothing (delete both lines). Fonts will be loaded via `<link>` tags in layout instead.

- [ ] **Step 2: Add animation keyframes to globals.css**

Append to the end of `app/globals.css`:
```css
@keyframes fadeSlideUp {
  from { opacity: 0; transform: translateY(12px); }
  to   { opacity: 1; transform: translateY(0); }
}

@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20%       { transform: translateX(-6px); }
  40%       { transform: translateX(6px); }
  60%       { transform: translateX(-4px); }
  80%       { transform: translateX(4px); }
}

.animate-fade-slide-up {
  animation: fadeSlideUp 0.4s ease both;
}

.animate-shake {
  animation: shake 0.35s ease both;
}
```

- [ ] **Step 3: Add font `<link>` tags to app/layout.tsx**

Replace the entire contents of `app/layout.tsx` with:
```tsx
import type { Metadata } from "next";
import "./globals.css";

export const metadata: Metadata = {
  title: "Vivid Care",
  description: "Workforce and roster management for care providers",
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <head>
        <link rel="preconnect" href="https://fonts.googleapis.com" />
        <link rel="preconnect" href="https://fonts.gstatic.com" crossOrigin="anonymous" />
        <link
          href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=Manrope:wght@700;800;900&family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&display=swap"
          rel="stylesheet"
        />
      </head>
      <body className="antialiased bg-surface text-on-surface">
        {children}
      </body>
    </html>
  );
}
```

- [ ] **Step 4: Verify icons render**

Open `http://localhost:3000` (or whichever port) and navigate to `/admin/dashboard`. Icons like "dashboard", "group", "badge" in the sidebar should now appear as actual symbols, not text. If they still show as text, hard-refresh with Ctrl+Shift+R.

- [ ] **Step 5: Commit**

```bash
git add app/globals.css app/layout.tsx
git commit -m "fix: load fonts via link tags to fix icon rendering"
```

## Task 3: Animate the login page

**Files:**
- Modify: `app/login/page.tsx`

- [ ] **Step 1: Add background gradient and card entrance animation**

In `app/login/page.tsx`, replace the outer wrapper div and card div:

```tsx

  <div className="w-full max-w-sm">
```
To:
```tsx

  <div className="w-full max-w-sm">
```


```tsx
<div className="bg-surface-container-lowest rounded-xl p-8 animate-fade-slide-up" style={{ boxShadow: '0 20px 40px rgba(25,28,30,0.08)', animationDelay: '0.1s' }}>
```

- [ ] **Step 2: Add shake animation to error message**

Change the error div:
```tsx
{error && (

  </div>

```
To:
```tsx
{error && (
  <div key={error} className="animate-shake flex items-center gap-2 text-error text-sm bg-error-container rounded-lg px-3 py-2">
    <span className="material-symbols-outlined text-base">error</span>
    {error}
  </div>

```

- [ ] **Step 3: Add hover effect to submit button**
